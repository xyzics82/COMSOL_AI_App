# CASE_SPEC — 페로브스카이트 IBC 2D (v2 첫 '진짜 2D' 케이스)

작성 2026-07-07 · 상태: 스파이크 진행 중 · 사용자 요청 원문 기반

## 1. 사용자 요청 (2026-07-07)

IBC(후면 깍지형 전극) 페로브스카이트 태양전지. n형 단위: 기판/Ag(200nm)/SnO2(20nm),
p형 단위: 기판/Al(200nm)/NiOx(10nm). 전극 폭 3um, n/p 간격 3um, 전극 길이 10um.
전극 폭·간격에 따른 성능 변화 + 셀 내부 전하 분포.

## 2. 모델 기하 (2D 단면, half-cell 대칭)

실제 소자는 [n전극-간격-p전극-간격] 반복 → 대칭점은 n전극 중앙과 p전극 중앙.
모델 도메인: x ∈ [0, L], L = W/2 + g + W/2 = W + g (W: 전극 폭, g: 간격, Wn=Wp=W 가정).
측면 경계 = 대칭(기본 zero-flux 절연이 곧 대칭 조건).

y(수직) 구조 — 사각형 5개로 타일링(전부 mappable → Map 메시):
- SnO2: [0, W/2] × [0, 20nm] (n쪽 CTL)
- NiOx: [L−W/2, L] × [0, 10nm] (p쪽 CTL)
- NiOx 위 흡수층 채움: [L−W/2, L] × [10, 20nm]
- 간격 흡수층: [W/2, L−W/2] × [0, 20nm] (간격부 바닥까지 흡수층 — 패시베이션 가정)
- 본체 흡수층: [0, L] × [20nm, 20nm + t_abs]

접점(경계, 이상 옴익): SnO2 바닥(x∈[0,W/2], y=0) = n접점(접지), NiOx 바닥 = p접점(V0).
Ag/Al 금속 200nm는 모델링하지 않음 — 이상 옴익 경계조건으로 대체 (한계 7절).
그 외 모든 외부 경계(윗면·간격 바닥·측면): 절연(기본) = 패시베이션/대칭.

## 3. 물리

- 반도체(semi) 2D + 면외 두께 d = 10um (사용자 제시 전극 길이).
- 광입사: 윗면(y = H)에서 수직 하향, Beer-Lambert (IBC는 전면에 전극이 없어 음영 없음
  — 가로 균일 조명이 정확한 가정). G_ph = 4π/(hc)∫ k(λ)F(λ)exp(−4πk(λ)(H−y)/λ)dλ,
  λ 300–850nm, 흡수층 도메인만.
- 도핑: SnO2 donor ND=1e19 (meskini2024), NiOx acceptor NA=1e18 (sahu2018).
  흡수층 진성 + SRH τ=38.7ns.
- 스터디: 평형 → V0 스윕 0→1.2V/0.02 (기존 확정 패턴 재사용).

## 4. 물성 (전부 출처 있음 — data/materials.json)

- MAPbI3 v1 (기존), NiOx v1 (Sahu 2018 세트), **SnO2 v1 신규**: Eg 3.44/χ 4.39/εr 9.0/
  Nc 2.2e18/Nv 1.8e19/μn 7.56/μp 0.1 — Meskini & Asgharizadeh, Sci. Rep. 14, 5723 (2024)
  Table 1 (ITO/SnO2/MAPbI3/Spiro 실험 J-V 재현으로 검증된 세트, 지수 원문 대조 완료).

## 5. 스윕·출력

- 그리드: x = 전극 폭 W [um] (기본 1,2,3,4,5), y = 간격 g [um] (기본 1,3,5). 기본 15솔브.
- 출력: 조합별 J-V/지표 + PCE·FF 히트맵 (기존 파이프라인 재사용).
- **2D 전하 분포**: 대표 조합에서 전자/정공 농도(semi.N, semi.P)를 V=0(단락)에서
  노드 평가 → matplotlib tripcolor → charge2d_*.png. [평가 API는 스파이크로 확정]

## 6. 검증 기준

1. 스파이크: W3/g3 1조합 수렴 + 전하분포 추출 성공.
2. 물리 정합성: g가 커질수록 가로 수송 거리 증가 → FF·PCE 감소 경향. W 의존성은
   수집 면적 대비 재결합 경로 균형 — 문헌상 간격 축소가 일반적으로 유리.
3. Jsc 상한: 1D 평판(같은 t_abs)의 Jsc 이하 (수집 손실 때문) — 1D 26.26 대비 확인.
4. 수렴 실패 0건 (기본 그리드).

## 6.5 스파이크 판별 기록 (2026-07-07)

- W3/g3에서 J-V가 2.0V까지 스윕 필요 — SnO2 장벽(ΔEc 0.49eV)+NiOx 전자차단(2.1eV)+
  계면 재결합 부재로 암전류가 극도로 억제되어 Voc가 Eg를 넘는 **이상화 인공물** 발생.
- 판별 실험: ETL을 C60(χ3.9, 무장벽)으로 바꾸면 1.3~1.4V에서 정상 교차 → 빌더 정상,
  물리 확인. C60-IBC Voc(≈1.35V) > 1D 평판(1.07V)인 것도 선택적 접점 효과로 정합.
- 결론: v0의 신뢰 지표는 **Jsc(수집 효율)와 기하 경향**. Voc·PCE 절대값은 v1(계면
  재결합 추가) 전까지 비교용.

## 6.6 정정 (2026-07-08) — Voc 인공물의 진범

6.5절의 "Voc > Eg 인공물" 원인 절반은 물리(장벽)가 아니라 **버그**였다:
`semi.create()`로 만든 SRH(tar1)에 selection을 지정하지 않으면 **빈 선택으로 시작**해
벌크 재결합이 아예 꺼져 있었다. `selection().all()` 수정 후 W3/g3(wave):
**Jsc 14.51 / Voc 1.039V / FF 0.436 / PCE 6.57%** — Voc 정상 범위 복귀.
낮은 FF는 여전히 SnO2 주입장벽(이상 옴익+계면재결합 부재) 영향으로 v1 과제 유지.
교훈: create()된 물리 기능은 반드시 선택을 명시할 것.

## 6.7 3D 확장 검증 (2026-07-08, v4)

- perovskite_ibc_3d(단위 셀: x=가로, y=수직, z=핑거 방향, Jsc 전용) 등록.
- 압출 극한(z_tip=Lz) 대조: 3D 14.869 vs 2D 14.511 mA/cm² — **2.4% 일치, 통과**
  (Map vs Sweep 메시 차이 수준).
- 첫 끝단 스터디: 핑거 4um 중 팁 3um(끝단 갭 1um) → Jsc 14.284 (**−3.9%**).
  셀의 25%가 핑거 너머인데 손실은 3.9% — 끝단 너머 캐리어도 z-확산으로 상당 수집됨
  (2D로는 볼 수 없는 3D 물리).

## 6.8 계면 재결합 v1 — IDL 등가층 방식 (2026-07-08, #29)

파라미터 `s_ifc_cms`(S, cm/s — Yang 2020 스윕 범위 1~1e5) 구현 기록:

- **1차 시도(경계 feature) 실패의 교훈 2건**:
  1. `TrapAssistedSurfaceRecombination`은 문서상(Semiconductor Module User's
     Guide p.116) **Insulation/Thin Insulator Gate/Insulator Interface/Metal
     Contact(Schottky) 경계 전용** — 반도체-반도체 내부 계면에는 선택을 지정해도
     기여 0 (실측: S=1e5에서도 지표 불변). 내부 계면 전용
     `Trap-Assisted Heterointerface Recombination`은 Continuity/Heterojunction을
     thermionic emission으로 바꿔야 하고(기본은 continuous quasi-Fermi, 그 옵션에선
     비활성) explicit trap 분포만 지원 → 물리 기저 변경이라 v2로 보류.
  2. **경계 Box 선택들의 Union은 physics `named()`에서 0개로 해석**된다
     (도메인 Union은 정상). 경계 합성이 필요하면 부품 선택 엔티티를 직접 합산해
     `selection().set()` 할 것.
- **채택(v1): IDL(계면 결함층) 등가 방식** — CTL 상면 바로 위 2 nm 흡수층 타일에
  SRH τ = d/S 부여 (S = d/τ 등가, 문헌 관용 근사). 검증된 API만 사용
  (도메인 TrapAssistedRecombination + Box 선택). s_ifc_cms=0이면 원래 5-타일
  지오메트리 그대로 (기준 보존).
- **검증 (W3/g3, wave_optics)**:
  | S (cm/s) | Jsc | Voc | FF | PCE |
  |---|---|---|---|---|
  | 0 (기준) | 14.511 | 1.0391 | 0.4357 | 6.57% |
  | 1000 | 14.549 | 1.0389 | 0.4358 | 6.58% |
  | 1e5 | 13.897 | 1.0014 | 0.4326 | 6.02% |

  단조·물리적 방향(S↑ → 전 지표↓), S=1000은 벌크 SRH 대비 부차적(Yang 2020 경향과
  일치), S=1000의 +0.3% Jsc는 IDL 타일 추가에 따른 메시 차이(수치 노이즈) 수준.
- **3D 확장 검증**: 같은 IDL을 3D(블록 겹침→파티션)로 적용. 압출 극한(z_tip=Lz),
  같은 메시(hmax 120nm)·같은 S=1000에서 **3D 18.900 vs 2D 19.009 = 0.6% 일치, 통과**.
  단 3D는 IDL(2nm 층) 포함 시 기본(자동) Sweep 메시가 평형 솔브부터 실패 →
  빌더에 hmax 120nm 자동 세분 안전망 추가.
- 잔여 한계: 간격부 바닥(절연 경계)의 표면 재결합은 미포함 — 이곳은 진짜 외부
  경계라 `TrapAssistedSurfaceRecombination` 적용 가능(문서상 Insulation 경계 지원),
  v1.1 후보. 팁 끝면(3D)도 동일하게 생략.

## 6.9 메시 수렴 연구 (2026-07-08, #29) — ⚠️ 기본 메시는 정량 불가

gap≥5um Newton 실패를 메시 세분(hmax 120nm)으로 해소하는 과정에서, **기본(auto)
메시가 Jsc를 크게 과소평가**해 온 것을 발견했다. W3/g3(wave, S=1000) 수렴 시퀀스:

| hmax | Jsc [mA/cm²] | Voc [V] | FF | PCE |
|---|---|---|---|---|
| 자동(기본) | 14.549 | 1.0389 | 0.4358 | 6.58% |
| 120 nm | 19.009 | 1.0505 | 0.4424 | 8.83% |
| 80 nm | 18.087 | 1.0483 | 0.4432 | 8.40% |
| 60 nm | 18.785 | 1.0498 | 0.4433 | 8.74% |
| 40 nm | 19.148 | 1.0505 | 0.4441 | 8.93% |

- 기본 메시: Jsc **−24%** (수직 방향이 성겨 표면 근처 고생성 구간 과소적분 + 수집
  구배 미해상). Voc/FF도 계통 편향.
- hmax≤120nm: Jsc 18.1~19.1 대역에서 진동(±3%, mapped 메시의 이산 요소 배치 효과),
  Voc/FF는 안정(1.048~1.051 / 0.442~0.444). 최세분(40nm) 기준값:
  **Jsc 19.15 / Voc 1.050 / FF 0.444 / PCE 8.93%** → 수집효율 19.15/22.50 ≈ 85%.
- **영향 범위**: 6.6절(14.51)·6.7절(3D 대조 14.87/14.51)·기존 15조합 그리드 수치는
  전부 기본 메시 값 = 계통적으로 낮음. **경향(상대 비교)은 유효하나 절대값은
  hmax≤80nm 결과만 인용할 것.** gap5가 gap3보다 높게 나온 "역전"도 이 아티팩트
  (gap5만 자동 세분이 걸렸었음).
- 정책: 케이스 폼에 mesh_hmax_nm 노출, **기본 80nm** (조합당 ~1.5분). 0=자동 메시는
  빠른 경향 훑기 전용. gap≥4um 자동 세분(120nm)은 유지(수렴 안전망).
- gap5 수렴(#29 목표 b): hmax 120nm에서 Newton 정상 수렴 확인 — W3/g5 완주
  (Jsc 17.18 / FF 0.362). 남은 진동 개선은 방향별 Distribution 제어(v2, API 미검증).

## 7. v0 한계 (정직 고지)

1. Ag/Al 일함수·접촉 장벽 미반영 (이상 옴익) — 금속 선택 효과는 안 보임.
2. 계면 결함(CTL/흡수층, 간격 바닥 표면 재결합) 미포함 — IBC에서 간격부 표면
   패시베이션이 실제로는 성능 좌우. v1에서 표면 재결합 속도 추가 권장.
3. 광학은 수직 Beer-Lambert (간섭·산란 무시) — 기존 케이스와 동일한 한계.
4. Wn=Wp 가정 (사용자 요청 '전극 폭' 단수). 필요시 분리 파라미터로 확장 가능.
5. SRH는 전 도메인 적용(τ 흡수층 값) — CTL은 고도핑이라 영향 미미하나 단순화임.
