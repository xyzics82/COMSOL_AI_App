# CASE_SPEC — 페로브스카이트 IBC 2D (v2 첫 '진짜 2D' 케이스)

작성 2026-07-07 · 상태: 스파이크 진행 중 · 사용자 요청 원문 기반

## 1. 사용자 요청 (2026-07-07)

IBC(후면 깍지형 전극) 페로브스카이트 태양전지. n형 단위: 기판/Ag(200nm)/SnO2(20nm),
p형 단위: 기판/Al(200nm)/NiOx(10nm). 전극 폭 3um, n/p 간격 3um, 전극 길이 10um.
전극 폭·간격에 따른 성능 변화 + 셀 내부 전하 분포.

## 2. 모델 기하 (2D 단면, half-cell 대칭)

실제 소자는 [n전극-간격-p전극-간격] 반복 → 대칭점은 n전극 중앙과 p전극 중앙.
모델 도메인: x ∈ [0, L], L = W/2 + g + W/2 = W + g (W: 전극 폭, g: 간격, Wn=Wp=W 가정).
측면 경계 = 대칭(기본 zero-flux 절연이 곧 대칭 조건).

y(수직) 구조 — 사각형 5개로 타일링(전부 mappable → Map 메시):
- SnO2: [0, W/2] × [0, 20nm] (n쪽 CTL)
- NiOx: [L−W/2, L] × [0, 10nm] (p쪽 CTL)
- NiOx 위 흡수층 채움: [L−W/2, L] × [10, 20nm]
- 간격 흡수층: [W/2, L−W/2] × [0, 20nm] (간격부 바닥까지 흡수층 — 패시베이션 가정)
- 본체 흡수층: [0, L] × [20nm, 20nm + t_abs]

접점(경계, 이상 옴익): SnO2 바닥(x∈[0,W/2], y=0) = n접점(접지), NiOx 바닥 = p접점(V0).
Ag/Al 금속 200nm는 모델링하지 않음 — 이상 옴익 경계조건으로 대체 (한계 7절).
그 외 모든 외부 경계(윗면·간격 바닥·측면): 절연(기본) = 패시베이션/대칭.

## 3. 물리

- 반도체(semi) 2D + 면외 두께 d = 10um (사용자 제시 전극 길이).
- 광입사: 윗면(y = H)에서 수직 하향, Beer-Lambert (IBC는 전면에 전극이 없어 음영 없음
  — 가로 균일 조명이 정확한 가정). G_ph = 4π/(hc)∫ k(λ)F(λ)exp(−4πk(λ)(H−y)/λ)dλ,
  λ 300–850nm, 흡수층 도메인만.
- 도핑: SnO2 donor ND=1e19 (meskini2024), NiOx acceptor NA=1e18 (sahu2018).
  흡수층 진성 + SRH τ=38.7ns.
- 스터디: 평형 → V0 스윕 0→1.2V/0.02 (기존 확정 패턴 재사용).

## 4. 물성 (전부 출처 있음 — data/materials.json)

- MAPbI3 v1 (기존), NiOx v1 (Sahu 2018 세트), **SnO2 v1 신규**: Eg 3.44/χ 4.39/εr 9.0/
  Nc 2.2e18/Nv 1.8e19/μn 7.56/μp 0.1 — Meskini & Asgharizadeh, Sci. Rep. 14, 5723 (2024)

## 5. 스윕·출력

- 그리드: x = 전극 폭 W [um] (기본 1,2,3,4,5), y = 간격 g [um] (기본 1,3,5). 기본 15솔브.
- 출력: 조합별 J-V/지표 + PCE·FF 히트맵 (기존 파이프라인 재사용).
- **2D 전하 분포**: 대표 조합에서 전자/정공 농도(semi.N, semi.P)를 V=0(단락)에서
  노드 평가 → matplotlib tripcolor → charge2d_*.png. [평가 API는 스파이크로 확정]

## 6. 검증 기준

1. 스파이크: W3/g3 1조합 수렴 + 전하분포 추출 성공.
2. 물리 정합성: g가 커질수록 가로 수송 거리 증가 → FF·PCE 감소 경향. W 의존성은
   수집 면적 대비 재결합 경로 균형 — 문헌상 간격 축소가 일반적으로 유리.
3. Jsc 상한: 1D 평판(같은 t_abs)의 Jsc 이하 (수집 손실 때문) — 1D 26.26 대비 확인.
4. 수렴 실패 0건 (기본 그리드).

## 6.5 스파이크 판별 기록 (2026-07-07)

- W3/g3에서 J-V가 2.0V까지 스윕 필요 — SnO2 장벽(ΔEc 0.49eV)+NiOx 전자차단(2.1eV)+
  계면 재결합 부재로 암전류가 극도로 억제되어 Voc가 Eg를 넘는 **이상화 인공물** 발생.
- 판별 실험: ETL을 C60(χ3.9, 무장벽)으로 바꾸면 1.3~1.4V에서 정상 교차 → 빌더 정상,
  물리 확인. C60-IBC Voc(≈1.35V) > 1D 평판(1.07V)인 것도 선택적 접점 효과로 정합.
- 결론: v0의 신뢰 지표는 **Jsc(수집 효율)와 기하 경향**. Voc·PCE 절대값은 v1(계면
  재결합 추가) 전까지 비교용.

## 7. v0 한계 (정직 고지)

1. Ag/Al 일함수·접촉 장벽 미반영 (이상 옴익) — 금속 선택 효과는 안 보임.
2. 계면 결함(CTL/흡수층, 간격 바닥 표면 재결합) 미포함 — IBC에서 간격부 표면
   패시베이션이 실제로는 성능 좌우. v1에서 표면 재결합 속도 추가 권장.
3. 광학은 수직 Beer-Lambert (간섭·산란 무시) — 기존 케이스와 동일한 한계.
4. Wn=Wp 가정 (사용자 요청 '전극 폭' 단수). 필요시 분리 파라미터로 확장 가능.
5. SRH는 전 도메인 적용(τ 흡수층 값) — CTL은 고도핑이라 영향 미미하나 단순화임.
