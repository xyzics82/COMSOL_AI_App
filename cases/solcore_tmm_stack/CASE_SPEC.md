# Solcore TMM 광학 케이스 — 사양·검증 기록

작성 2026-07-08. 엔진: Solcore (파이썬 in-process, 동글·외부 프로그램 불필요).

## 1. 문제 정의

공기 / MAPbI3(t_abs) / 공기 평판, 수직입사 TE — **COMSOL 파동광학 v3
(backend/wo_optics.py)과 동일한 문제**. 같은 데이터(mapbi3_nk, am15)를 사용한다.

산출: R/A/T(λ), AM1.5 가중 G(depth), Jsc,opt(광학 상한).

- ∫a(z)dz vs A(λ) 최대 편차 1e-3 (흡수 프로파일 정합).

## 3. 부수 발견 — COMSOL 광학 λ그리드 권고

λ 15점(현 wo_optics 기본)은 간섭 프린지를 언더샘플링해 **Jsc,opt를 ~2.8% 과대**
(22.50 vs 수렴 21.88). 대책: 이 케이스(수 초)로 수렴 Jsc,opt를 얻어 COMSOL 결합
케이스의 광학 상한 해석에 참고. wo_optics nlam 상향은 λ당 ~3s 비용 문제라 보류
(15점 유지 시 상한이 3% 후하게 잡히는 것만 인지).
