# v3 업그레이드 사양 — IDE 광학·전기 결합 (Wave Optics + Semiconductor)

작성 2026-07-07 · 근거: Miglani 2025 (Small, Fig 2b IDE), Yang 2020 (EES 13, 1753 — QIBC 소자물리)

## 0. 목표

Miglani Fig 2b의 IDE(동일 평면 깍지 전극) 구조를, Yang 2020처럼
(a) 파동 광학으로 광자 거동(간섭 포함)을 풀어 위치별 광생성 G(x,y)를 얻고
(b) 그것을 2D 반도체 모델에 주입해 전하 거동까지 잇는 결합 시뮬레이션.

현재 보유: IDE 단면 전기 모델 = IBC 2D 빌더(완성, 전하분포 맵 포함).
새로 만들 것: Wave Optics 파트와 결합 파이프라인.

## 1. 단계 (각 단계 = 스파이크 → 확정 → 구현, 기존 검증 패턴)

- **O-1 Wave Optics API 확정** [이번]: 설치 예제 plasmonic_wire_grating.mph 로드·전체
  덤프 (v0의 si_solar_cell_1d 덤프와 같은 전략). 확정 대상: ewfd 물리 생성,
  주기 Port(평면파 입사)·Floquet 경계, 재료 복소 굴절률(n, k) 설정, 주파수 스터디,
  S-파라미터(R/T)·흡수 손실(Qh) 평가.
- **O-2 슬랩 검증**: 공기/MAPbI3(800nm)/공기 단층 슬랩, λ 3점 — R/T/A를 전달행렬(TMM)
  파이썬 계산과 대조 (±수% 일치 = 광학 빌더 검증 기준). MAPbI3 n·k는 기존
  mapbi3_nk 데이터(n열 포함) 사용.
- **O-3 G(x,y) 합성**: λ 15~20점(300–850nm) 파라메트릭 스윕 → 각 λ의 Qh(x,y) 노드
  평가 → 파이썬에서 G(x,y) = Σ_λ Qh/(ħω) × AM1.5 가중 → (x,y,G) 테이블.
- **O-4 결합**: IDE 기하(IBC 빌더 재사용)에서 semi의 UDG에 G(x,y) 2D 보간함수 주입
  (Beer-Lambert 대체). 광학 기하 = 전기 기하 동일 단면.
- **O-5 분석 세트(Yang식)**: |E|²(대표 λ) 맵, G(x,y) 맵, 전하밀도 맵(기존), J-V/지표,
  (v1+) 재결합 손실 맵.

## 2. 새 데이터 요구 (임의값 금지 — 전부 refractiveindex.info 등 출처)

- n,k 추가 필요: SnO2, NiOx (CTL — 얇지만 광학 경로에 있음), 전극 금속(Au/Ag/Al).
- v0 광학 단순화(정직 고지): 유리 기판·ITO 생략, 전극 반사는 후면이라 2차 효과 —
  공기/흡수층/CTL 구간만. 금속 반사 포함은 v1.
- 데이터 레지스트리에 sno2_nk, niox_nk 항목 추가 예정 (자동 다운로드 후보 URL 포함).

## 3. 케이스 (완성 시)

- id: perovskite_ide_opto (가칭), builder: "ide_opto"
- 그리드: W × gap (IBC와 동일), 출력: 지표 + G맵 + |E|²맵 + 전하맵
- 파장 분해능·λ 구간은 폼 파라미터 (기본 300–850nm, 20점)

## 4. 정직한 경계

1. 파장 루프 × 기하 그리드 = 계산량 곱 (예: λ20 × 15조합 = 300 광학해).
   v0는 광학을 기하당 1회(수직입사, TE) — 편광/각도 의존은 v1.
2. Yang의 광 재활용(photon recycling)·이온 이동은 범위 밖 (명시적 제외).
3. Voc 이상화 문제는 IBC와 동일 — 계면 재결합(v1)과 함께 해결.
4. 3D IDE(손가락 길이 방향)는 2D 단면 근사 — 주기 단면이 IDE의 표준 모델링.
