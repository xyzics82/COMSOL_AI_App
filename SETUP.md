# 설치·실행 (웹앱 방식 — 터미널 불필요)

## 필요한 것

- **Python 3.10~3.13** (3.12 권장). ⚠️ **3.14는 사용 불가** — COMSOL 연결 라이브러리(JPype)가
  아직 지원하지 않을 가능성이 높음. start.bat이 3.12→3.13→3.11→3.10 순으로 자동 탐색하며,
  하나도 없으면 3.12 설치를 안내함: https://www.python.org/downloads/
  (설치 시 "Add python.exe to PATH" 체크)
- COMSOL 6.4 (확인 완료: `C:\Program Files\COMSOL\COMSOL64\Multiphysics_copy1`)

## 실행

**`start.bat` 더블클릭.** 이것이 전부입니다.

- 최초 1회: 가상환경 생성 + 패키지 설치 (수 분)
- 브라우저에서 http://127.0.0.1:8712 자동 오픈
- 서버 창(검은 콘솔)을 닫으면 앱이 종료됩니다

## 첫 사용 순서 (UI 탭)

1. **① 환경 점검 실행** — COMSOL 세션 기동 + Si 예제 솔브 + 모델 구조 덤프 (수 분).
   완료 후 ④ 탭에서 **로그 복사** → Claude에게 붙여넣기 (다음 개발 단계의 입력값)
2. **② 데이터 준비** — "자동 다운로드 시도" 클릭. 실패하면 안내된 페이지에서 CSV를 받아 업로드
3. **③ Si 데모 실행** — 파이프라인 전체(솔브→J-V→지표→플롯) 검증. 로그의 지표가
   Voc≈0.61V, Isc≈33mA 근처인지 확인
4. **③ 페로브스카이트 실행** — 초안이므로 오류가 나면 로그 복사 → Claude 회신 → 수정 반복

## 5. 자연어 입력 — 두 가지 방식

**방식 A: 복붙 모드 (기본, 키·추가비용 불필요)**
③ 탭에서 문장 입력 → "① 프롬프트 생성+복사" → 구독 중인 챗(claude.ai, ChatGPT 등)에 붙여넣기
→ AI의 JSON 답변을 "②"에 붙여넣고 적용. 구독제만 있으면 됩니다.

**방식 B: API 자동 모드 (Anthropic API 키)**

1. https://console.anthropic.com 에서 계정 생성/로그인
2. **Billing**에서 결제 수단 등록 또는 크레딧 충전 (최소 $5 수준이면 충분 — 요청당 1~2센트 미만)
3. **API Keys** → **Create Key** → 키 복사 (`sk-ant-...`)
4. 프로젝트 폴더의 `.env.example`을 복사해 같은 폴더에 **`.env`** 로 저장하고 키 붙여넣기:
   `ANTHROPIC_API_KEY=sk-ant-...`
5. 콘솔 창 닫고 `start.bat` 재실행 (anthropic 패키지 자동 설치 + 키 로드)
6. ③ 탭 상단 자연어 입력창이 활성화됨 ("모델: claude-sonnet-5" 표시 확인)

⚠️ .env 파일은 결제 수단과 같으니 절대 공유하지 마세요.

## 문제 해결

- `start.bat` 창에 빨간 오류 → 창 내용 복사해서 Claude에게
- UI의 작업이 failed → ④ 탭 로그 복사해서 Claude에게
- 포트 충돌(8712 사용 중) → start.bat 마지막 줄의 `--port 8712`를 다른 번호로
